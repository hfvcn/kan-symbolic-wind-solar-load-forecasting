# 文献仓库（可直接引用｜已尽量选用一手来源）

> 目标：为后续正式论文写作准备“可直接引用”的文献池。  
> 收录标准（尽量满足其一）：**DOI** / **arXiv ID（含 10.48550/arXiv… DOI）** / **出版社或权威数据库页面（IEEE/Elsevier/ACM/Springer/PMC/OSTI/NREL 等）**。  
> 不收录（默认）：仅 ResearchGate “Request PDF/figure”、博客/二手解读页（可作为背景阅读，但不建议写进参考文献）。
