### 🎨 附：顶会图表的美学与执行纪律（防“廉价感”指南）

要做到“不过分花哨”，质感来源于极简。请严格遵守以下绘图代码规范：

1.  **绝对禁止位图（截图格式）**：所有实验图必须在 Python 中通过 `plt.savefig('name.pdf')` 导出为 **PDF 矢量图**再插入 LaTeX。放大 1000% 边缘必须依旧锐利。图里出现锯齿是直接被判定为“水文”的第一视觉诱因。
2.  **一键提升科研质感**：抛弃 matplotlib 和 excel 难看的默认样式。强烈建议在 Python 环境中 `pip install SciencePlots`，并在代码首行加入：
    ```python
    import scienceplots
    plt.style.use(['science', 'ieee', 'no-latex']) 
    ```
    只需这一行，你的坐标轴向内刻度、精细边框和干净底色就能瞬间达到顶级期刊标准。
3.  **克制且固定的调色板**：不要用高饱和的大红大蓝。使用色盲友好色系。**全篇固定语义色彩**：只要提到滞后项就是深灰，Wind 永远是莫兰迪蓝，Solar 永远是琥珀黄，Load 永远是砖红。这会建立强大的视觉潜意识。
4.  **图注自包含（Self-contained Captions）**：图的标题（Caption）绝不能只有半句话（如“图X：阻断结果”）。必须包含三要素：说明图的主题 + 解释 X/Y轴与图例 + **直接点明结论**（*“如图所示，干预后特征 VER 显著反弹，直观验证了捷径压制机制...”*）。让不读正文的评委看图也能领会核心精神。
