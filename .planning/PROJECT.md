# 基于符号回归的风光耦合负荷预测模型可解释性研究

## What This Is

毕业设计项目：使用柯尔莫哥洛夫-阿诺德网络符号回归（KAN-SR）从风光负荷耦合数据中自动提取具有物理意义的可解释数学公式。项目聚焦于可解释性，旨在解决深度学习"黑盒"问题，为电力系统调度提供透明、可审计的预测模型。

## Core Value

**从数据中提取人类可理解的物理公式** — 不仅预测准确，更要让电网工程师能看懂、能验证、能信任预测逻辑。

## Context

**研究背景：**
- 高比例可再生能源电网面临强随机性和非线性耦合
- 深度学习（LSTM、Transformer）虽精度高但缺乏可解释性
- 电网调度需要透明、可审计的预测模型

**技术选型依据：**
- KAN-SR 优于传统 GP-SR（如 PySR）：避免维数灾难，与深度学习工具链兼容
- KAN-SR 优于 EQL：数值稳定性更好
- KAN-SR 配合 PIKAN：可融入物理先验约束

**数据来源：**
- 主数据集：ARPA-E PERFORM (NREL) — 美国多个 ISO 的风光负荷同步数据
- 备选：斯坦福 SUNSET 数据集（含云图等多模态数据）

**计算环境：**
- Google Colab (云 GPU)
- Python 生态，框架灵活选择

## Constraints

- **计算资源**: Google Colab — 需要考虑显存限制和会话时长
- **时间**: 尽快达到可用水准 — 优先 MVP，后续可深入
- **核心指标**: 可解释性优先于纯精度 — 能提取有意义的公式比极致精度更重要
