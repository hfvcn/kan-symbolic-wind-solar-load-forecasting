# Phase 05 Plan 01: Evaluation Framework

## Goal

构建一个可复用的评估框架，能对不同 run 的产物进行统一汇总：
- Accuracy：RMSE/MAE/R²（test）
- Complexity：公式 node_count / PySR complexity / Torch param_count / KAN edge_pruned_ratio
- Compute time：started_at ~ completed_at（payload）
- Pareto optimality（rmse vs complexity）
- Seasonal breakdown（DJF/MAM/JJA/SON）
