Create `src/data/split.py` implementing strict chronological splitting and normalization:

1. `chronological_split(df: pd.DataFrame, train_ratio: float = 0.7, val_ratio: float = 0.15, gap_steps: int = 48) -> tuple[pd.DataFrame, pd.DataFrame, pd.DataFrame]`:
   - Assume df is sorted by DatetimeIndex (verify and raise if not)
   - Calculate split indices: train_end = int(n * train_ratio), val_end = int(n * (train_ratio + val_ratio))
   - Apply gap_steps between splits to prevent lag feature leakage:
     - train = df.iloc[:train_end]
     - val = df.iloc[train_end + gap_steps:val_end]
     - test = df.iloc[val_end + gap_steps:]
   - Log split sizes and date ranges
   - Return (train, val, test)

2. `drop_initial_lag_rows(df: pd.DataFrame, n_lags: int = 48) -> pd.DataFrame`:
   - Drop first n_lags rows that have NaN from lag feature creation
   - Return cleaned df

3. `normalize_features(train: pd.DataFrame, val: pd.DataFrame, test: pd.DataFrame, exclude_cols: list[str] = None) -> tuple[pd.DataFrame, pd.DataFrame, pd.DataFrame, dict]`:
   - exclude_cols defaults to target columns (caller specifies)
   - Fit StandardScaler on train features only (prevent data leakage)
   - Transform train, val, test using fitted scaler
   - Build scaler_params dict: {"mean": list, "scale": list, "feature_names": list}
   - Return (train_scaled, val_scaled, test_scaled, scaler_params)

4. `save_scaler_params(scaler_params: dict, output_path: str) -> str`:
   - Write scaler parameters as JSON
   - Include metadata: creation timestamp, sklearn version
   - Return path for logging

5. `save_splits_to_parquet(train: pd.DataFrame, val: pd.DataFrame, test: pd.DataFrame, output_dir: str, timestamp: str = None) -> dict[str, str]`:
   - If timestamp not provided, generate from current UTC time
   - Save each split as {split}_{timestamp}.parquet with snappy compression
   - Create {split}_{timestamp}_meta.json with row count, column list, date range
   - Return dict mapping split name -> parquet path

6. `load_splits_from_parquet(output_dir: str, timestamp: str = None) -> tuple[pd.DataFrame, pd.DataFrame, pd.DataFrame]`:
   - If timestamp not provided, find latest by filename sort
   - Load train, val, test Parquet files
   - Return (train, val, test)

   ```python
   def step_complete(run_dir: str, step_name: str) -> bool:
       marker = f"{run_dir}/.step_{step_name}_done"
       return os.path.exists(marker)

   def mark_step_complete(run_dir: str, step_name: str, volume):
       marker = f"{run_dir}/.step_{step_name}_done"
       with open(marker, "w") as f:
           f.write(datetime.now(timezone.utc).isoformat())
       volume.commit()

   def cleanup_partial_output(output_path: str):
       """Remove partial output file/directory on step failure."""
       if os.path.isfile(output_path):
           os.remove(output_path)
       elif os.path.isdir(output_path):
           shutil.rmtree(output_path)
   ```
