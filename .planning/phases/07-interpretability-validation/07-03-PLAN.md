# Phase 07 Plan 03: PySR Cross-Validation (KAN Seeds)

## Goal

用 PySR 对 KAN-SR 的关键子表达式做独立交叉验证（EVAL-08）：
- 从 KAN 提取公式中自动生成若干“seed 特征”（子表达式值）
- 将 seed 特征作为额外输入列喂给 PySR，观察其是否选择这些结构或复现相似形式

注意：该方案不依赖 PySR 内部“初始种群”接口，能稳定复现并可在论文中解释为“独立验证”。

## Tasks

- [ ] 新增 `src/eval/seed_features.py`：从 SymPy 表达式中提取可计算的 seed 子表达式
- [ ] 扩展 `modal_jobs/pysr_baseline.py`：可选 `--seed-from-symbolic-run <run_id>`，将 seed 特征加入 X
- [ ] 新增本地脚本 `scripts/compare_kan_pysr.py`：对比 PySR 方程与 KAN seed（相似度/匹配情况）
