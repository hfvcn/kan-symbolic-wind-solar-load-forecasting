# Phase 02 Plan 02: Prune + Refit + Sparsity Report

## Goal

在训练结束后完成稀疏化闭环：
- 复合正则化（lamb / lamb_l1 / lamb_entropy / lamb_coef / lamb_coefdiff）
- 剪枝（`KAN.prune(node_th, edge_th)`）并输出稀疏度统计
- 在多个剪枝阈值候选中选择“尽量高稀疏 + RMSE 不退化太多”的方案
- 剪枝后用 LBFGS 做短 refit
- 输出 `artifacts/sparsity.json` 与 `artifacts/feature_importance.csv`
