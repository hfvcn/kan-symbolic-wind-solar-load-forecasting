# Phase 02 Plan 01: KAN Training (small slice) + checkpoints

## Goal

实现一个可以在 Modal 上真正跑通的 KAN 训练作业：
- 读取 Phase 1 的 Parquet train/val/test
- 构建 KAN.fit 所需 dataset dict（train_input/train_label/test_input/test_label）
- 训练过程中产出 `payload.json` / `metrics.csv` / `checkpoint/`
- 采用“分段训练 + 每段 commit”的方式，满足“每 15 分钟可恢复”的约束（近似实现：每 chunk 写 checkpoint 并 commit）

## TDD / Verification (先小规模验证)

1. 写一个最小可跑的单元测试：
   - 用合成数据训练一个小 KAN
   - 验证 forward 输出维度正确
   - 验证 sparsity 统计结构正常

2. 仅在上述测试通过后，再实现 Modal Job 端到端写盘逻辑。
